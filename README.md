A modern Tic Tac Toe game built with HTML, CSS, and JavaScript. The project features a glassmorphism-inspired interface, responsive layout, live score tracking, animated interactions, and automatic win detection.

## Overview

This browser-based Tic Tac Toe game provides a clean and modern user experience while demonstrating core front-end development concepts such as DOM manipulation, event handling, game state management, and responsive design.

## Features

### Gameplay

- Interactive 3×3 game board
- Two-player mode (Player O vs Player X)
- Automatic turn switching
- Win detection for rows, columns, and diagonals
- Draw detection
- Real-time game status updates

### User Interface

- Modern glassmorphism design
- Gradient background with blur effects
- Live scoreboard
- Winning combination highlighting
- Draw-state visual feedback
- Tile click animations
- Responsive layout for desktop and mobile devices

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Inter)

## Getting Started

Clone the repository:

```bash
git clone https://github.com/zippynx/tic-tac-toe.git
```

Open `index.html` in your preferred browser.

No installation or additional dependencies are required.

## Game Rules

1. Players take turns placing O and X on the board.
2. The first player to align three marks horizontally, vertically, or diagonally wins.
3. If all cells are filled and no winner is found, the game ends in a draw.
4. Click **Play Again** to start a new round.
