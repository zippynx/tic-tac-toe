# Tic Tac Toe

A browser-based Tic Tac Toe game developed with HTML, CSS, and JavaScript. The game features dynamic board generation, turn-based gameplay, and automatic win detection.

## Features

- Interactive 3×3 game board
- Player turn switching (X and O)
- Row, column, and diagonal win detection
- Winning combination highlighting
- Lightweight and dependency-free

## Technologies

- HTML5
- CSS3
- JavaScript

## Getting Started

Simply open `index.html` in your browser to play.

## Project Purpose

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and game logic implementation.
